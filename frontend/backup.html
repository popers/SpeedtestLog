Jak skonfigurować Client ID/Secret?
  1. Wejdź na Google Cloud Console.
  2. Utwórz projekt i włącz API "Google Drive API".
  3. W "Credentials" utwórz "OAuth Client ID" (Typ: Web application).
  4. Dodaj ten adres w "Authorized redirect URIs":
    .../api/backup/google/callback
